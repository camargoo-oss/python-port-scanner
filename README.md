# 🔍 Python Port Scanner

A network port scanner built with Python, developed as a cybersecurity learning project.

## 📌 Versions

| Version | Description | Key Feature |
|---------|-------------|-------------|
| V1 | Basic scanner | TCP socket connection |
| V2 | CLI tool | Arguments + error handling |
| V3 | Fast scanner | Multithreading (30x faster) |
| V4 | Nmap integration | Service & version detection |

## 🛠️ Technologies
- Python 3
- socket, threading, argparse, python-nmap

## 🚀 Usage
```bash
python scanner_V3.py <target> <start_port> <end_port>
python scanner_V3.py 192.168.1.1 1 1024
```

## 🎯 Skills Demonstrated
- TCP/IP networking fundamentals
- Python scripting for security tools
- Multithreading and performance optimization
- Network reconnaissance concepts
